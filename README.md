# Socialreaders

This repository serves as my portfolio project. It features a frontend web app built with Next.js and a backend built with ASP.NET Core.

## Features

- [x] Component architecture using Atomic Design and Feature-Driven Development
- [ ] Component documentation and development using Storybook
- [x] CSS Variables exported from Figma using Design Tokens Transformer
- [x] Accessible components using React Aria
- [x] Smooth page transitions using View Transitions API
- [ ] Internationalization using next-i18next
- [ ] TypeScript type generation from OpenAPI specifications
- [ ] Client-side form validation using Zod
- [ ] Database using PostgreSQL
- [ ] Authentication using NextAuth.js
- [ ] Add and manage custom books in user library
- [ ] Code formatting and linting with custom ESLint rules
